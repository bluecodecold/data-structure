**串的存储结构**：
- 顺序存储
- 链式存储

**静态数组的实现（定长顺序存储）**
```c
# define MAXLEN 255 // 最大长度

typedef struct{
    char ch[MAXLEN]; // 串的存储结构
    int length; // 串的长度
}SString;
```

**动态数组的实现（堆分配存储）**
```c:
#define MAXLEN 255

typedef struct{
    char *ch; // 串的存储结构
    int length; // 串的长度
}HString;

HString S;
S.ch = (char *)malloc(MAXLEN * sizeof(char)); // //按串长分配存储区，ch指向串的基地址 用完需要free
s.length = 0; //初始长度为0
```
